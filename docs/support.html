Assistance - Azar

Derniere mise a jour : 7 juillet 2026

Azar est une application simple de pile ou face personnalisable. Cette page regroupe les informations utiles pour obtenir de l'aide.

Utilisation

Appuyez sur le bouton Lancer pour faire tourner la piece. Vous pouvez aussi choisir une prediction, personnaliser les libelles Pile et Face, et ajouter une image locale pour chaque face.

Images et donnees locales

Les images choisies restent sur votre appareil. Si vous voulez les retirer, ouvrez la personnalisation, appuyez sur Retirer pour chaque face, puis enregistrez.

Reinitialisation

Le bouton Reinitialiser efface l'historique de la session en cours. Pour supprimer les reglages, les images personnalisees et les statistiques locales, ouvrez l'ecran Confidentialite dans l'app puis utilisez Effacer les donnees locales. Vous pouvez aussi desinstaller l'application.

Confidentialite

Azar fonctionne sans compte, sans publicite, sans analytics et sans collecte de donnees. Consultez la politique de confidentialite.
