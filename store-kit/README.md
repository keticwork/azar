# Store kit Azar

Kit prepare pour remplir App Store Connect et Google Play Console.

## Dossiers

- `images/apple-app-store/` : captures iPhone 6,5 pouces en PNG `1242 x 2688`.
- `images/apple-app-store/upload-ready-iphone-65-v2/` : captures reelles TestFlight redimensionnees pour App Store Connect, recommandees pour la soumission.
- `images/google-play/` : feature graphic Google Play `1024 x 500`.
- `icons/` : icones store exportees depuis les assets de l'app.
- `sources/` : pages HTML source utilisees pour generer les visuels.

## URLs publiques

- Politique de confidentialite : https://keticwork.github.io/azar/privacy-policy.html
- Assistance : https://keticwork.github.io/azar/support.html

## Notes

Les captures sont des visuels de listing prepares depuis l'interface reelle d'Azar. Pour la soumission finale, remplacer par des captures reelles iPhone si Apple demande une stricte correspondance pixel-par-pixel avec l'app installee.

Ordre conseille pour App Store Connect > iPhone 6,5 pouces :

1. `images/apple-app-store/upload-ready-iphone-65-v2/01-pile-par-defaut.png`
2. `images/apple-app-store/upload-ready-iphone-65-v2/02-personnalisation-images.png`
3. `images/apple-app-store/upload-ready-iphone-65-v2/03-resultat-personnalise.png`
4. `images/apple-app-store/upload-ready-iphone-65-v2/04-statistiques-historique.png`
5. `images/apple-app-store/upload-ready-iphone-65-v2/05-confidentialite.png`

Ne pas utiliser de capture contenant `TestFlight`, `Expo Go`, une barre de retour beta ou des images sous licence/marques tierces. Pour montrer les images personnalisees, utiliser une image neutre ou possedee par le developpeur.
